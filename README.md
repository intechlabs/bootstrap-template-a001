![intechlabs web solutions bootstrap template preview](https://i.imgur.com/CfqqpdD.gif)

#### To compile from SASS
```sh
yarn run sass
```

#### To compile from SASS and watch
```sh
yarn run sass:watch
```

#### To create a build using postcss ( autoprefixer, purgecss, & cssnano )
```sh
yarn run build
```
